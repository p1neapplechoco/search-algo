# search-algo
CSC140003 - Project 1 

## Setup

Go to the desired directory, clone the repository:
```
git clone https://github.com/p1neapplechoco/search-algo
cd search-algo
```

Virtual Environment Setup:
```
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate  # Windows
```

Install Dependencies:
```
pip install -r requirements.txt
```

Activate virtual environment:
```
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate  # Windows
```


## Load src files
Use `setup.py` using *setuptools* library to load source files as packages.

To install the package, run:
```
python setup.py install
```

To uninstall the package, run:
```
python setup.py clean
```


## Run the project
To run the project, which is notebooks under `notebooks/` folder, use Jupyter Notebook or Jupyter Lab:

```
jupyter notebook
jupyter lab
```

Then open the desired notebook file and run the cells.
